Markey is a simple bookmark manager which saves and opens websites using keyboard shortcuts.

- To bookmark an active website, use the shortcut Ctrl+Shift+B on the browser.

- To open a bookmark, Press Ctrl+Shift+Alt+B anytime. This will open a list of your saved sites. You can choose the desired site and enter its index in the input box.


Pros:
1. Everything saved locally.
2. Bookmarks can be managed irrespective of browsers used.
3. No limit to the number of bookmarks.
